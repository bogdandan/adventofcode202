# INSTALL
```bash
    yarn
```

# RUN A PACKAGE
```bash
    yarn workspace day1 run start:dev
```

# PROVIDE A FILE ( linux / macos )
```bash
    export FILE=input
```
# CLEAN
``` bash
    yarn workspaces clean
```

# BUILD
```bash
    yarn workspaces run build
```
or 
```bash
    yarn workspace day1 run build
```
